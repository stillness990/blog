## 为什么选择 MkDocs Material

MkDocs Material 很适合个人开发者：

- 使用 Markdown 写作，简单稳定
- 内置全文搜索
- 支持博客、标签、分类、代码高亮
- 可以部署到 GitHub Pages
- 运维成本低
- 适合长期沉淀知识库
