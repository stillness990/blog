# 知识库

知识库用于沉淀长期可复用的技术知识。和博客不同，知识库更关注结构化、可查找和可持续更新。

## 主题入口

- [AI Agent](../topics/ai-agent/index.md)
- [Claude Code](../topics/claude-code/index.md)
- [OpenAI Codex](../topics/openai-codex/index.md)
- [自动化工具](../topics/automation/index.md)

## 维护方法

1. 博客记录完整过程
2. 笔记记录零散知识
3. 知识库沉淀稳定结论
4. 项目页记录实战经验
