# 位置推送系统

这是一个用于记录和展示位置推送系统开发过程的项目页面。

## 背景

记录 Android 端位置采集、后端 API 接收、Web 端展示和部署过程。

## 技术栈

- Android / Kotlin
- FastAPI 或 Node.js API 服务
- Web 前端
- Docker / Docker Compose
- GitHub

## 核心模块

- Android 位置采集
- 设备注册与认证
- 位置数据上传
- 后端数据存储
- Web 地图展示
- 历史轨迹查询
